# 前言

---

从小到大,一直以来,听到的关于程序员的印象都是经典格子衫,双肩包的形象.在我准备成为计算机大家族中的一员的时候,我觉得程序员应该不仅仅是人们印象中的呆板的形象.所以我又有一个目标: 要成为精致的人里最会敲代码的,会敲代码里最精致的. 这是我刚入这个行业之初的想法.

事实上,行业里早就有很多气质形象俱佳的大佬了!作为一位励志要成为一位攻城狮的程序员.在提升自己专业技能之余,也想花点时间用于个人形象气质的塑造提升中...

网络上,很少有系统地教一位男生来提升形象气质的文章教程.所以,我就自己把自己平时看到的关于形象提升的文章摘录下来,用来严格要求自己,希望自己在成为攻城狮的道路上一步一步前进！
